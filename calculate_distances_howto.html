Calculate distances using a JavaScript programme – how it works

This calculation is made by means of the side cosine theorem, since a spherical (Eulerian) triangle can be formed with point 1, point 2 and the north pole. The distance between point 1 and point 2 is therefore the great circle arc g. So the formula is:

cos g = sin φ1 sin φ2 + cos φ1 cos φ2 cos Δλ

(φ: Latitude / Δλ: Difference of longitudes)

From this the kilometre value can be calculated using the following formula:

Distance in kilometres = r g 2 π / 360°

where the radius r is assumed to be the mean radius of the Earth, 6371.0088 km. [1]

The given coordinates are converted into decimal degrees by means of the function min2dec and then converted into radians by means of the function deg2rad, because the methods sin and cos of the JavaScript object Math expect the data in radians. The method acos (arc cos), on the other hand, returns the result in radians, which is converted back into degrees via the function rad2deg. Finally, the result is rounded to 3 decimal places using the round method of the Math object.
